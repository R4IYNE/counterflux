# Phase 08.1 Plan 01: Add-Card Panel Polish (Dropdown Scroll + Re-Open Affordance) Summary

Two pure-CSS / lightweight-DOM polish fixes from the Phase 8 human-UAT walkthrough — both touch the LHS Treasure Cruise add panel surface. Dropdown scroll containment now lives on the dropdown container (FOLLOWUP-1); the collapsed-panel re-open affordance is now an unmistakable 48px primary-accent button with an inline 'OPEN PANEL' label (FOLLOWUP-2).

## What Shipped

### Task 1 — Dropdown scroll containment (FOLLOWUP-1)

**Problem:** Typing a query that returned 8+ matches in the LHS add panel rendered all rows, but the parent `<aside class="tc-panel-column">` declared `overflow-y: auto`, which clipped the absolutely-positioned dropdown when it overflowed the aside's content box. Users saw a scrollbar but couldn't reach rows below the panel boundary.

**Fix:**

- `<aside class="tc-panel-column">` switched from `overflow-y: auto` to `overflow: visible` so descendants can escape the content box.
- New inner `<div class="tc-panel-body">` wrapper now owns the panel body scroll (`flex: 1; min-height: 0; overflow-y: auto; padding: 24px; display: flex; flex-direction: column; gap: 16px;`). The padding (24px) and gap (16px) that previously lived on the aside moved here so they no longer apply to the outermost flex container.
- Dropdown wrapper's own `max-height: 280px; overflow-y: auto; position: absolute` declaration unchanged — already correct in isolation; the bug was never the dropdown.
- Body wrapper opens immediately after the aside opening tag and closes immediately before `</aside>`, preserving the existing internal markup unchanged.

### Task 2 — Re-open affordance redesign (FOLLOWUP-2)

**Problem:** The original 32×32 ghost-bordered chevron at top-left of the Treasure Cruise grid (when the LHS panel was collapsed) failed the user's "I can't see this" test during human-UAT.

**Fix:**

- New `.cf-panel-reopen` utility class in `main.css`: 48px hit-target, `var(--color-primary)` background, glow-blue box-shadow, JetBrains Mono 11/700 uppercase label, paired chevron_right glyph at 24px.
- Hover boosts box-shadow blur (12px → 20px). Focus-visible adds 2px primary text outline + 2px offset.
- `src/screens/treasure-cruise.js` re-open button now carries `class="cf-panel-reopen"` only — all inline styles and onmouseenter/onmouseleave handlers dropped. Markup added the inline `<span>OPEN PANEL</span>` label.
- Reduced-motion: extended the existing `@media (prefers-reduced-motion: reduce)` block (lines 147-163 of `main.css`) to include `.cf-panel-reopen` in the selector list.
