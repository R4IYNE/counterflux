    1. Install dependencies: `npm install papaparse chart.js`

    2. Modify `src/db/schema.js`: Add version(2) AFTER the existing version(1) block:
    ```javascript
    db.version(2).stores({
      cards: 'id, name, oracle_id, set, collector_number, cmc, color_identity, type_line, [set+collector_number]',
      meta: 'key',
      collection: '++id, scryfall_id, category, foil, [scryfall_id+foil], [scryfall_id+category]'
    });
    ```
    The collection table fields are:
    - `++id` auto-incrementing primary key
    - `scryfall_id` (string) FK to cards.id
    - `quantity` (number, NOT indexed, stored as property)
    - `foil` (number 0 or 1 -- NEVER boolean, IndexedDB cannot index booleans)
    - `category` (string: 'owned' or 'wishlist')
    - `added_at` (ISO string, NOT indexed, stored as property)
    Compound indexes: `[scryfall_id+foil]` for duplicate detection, `[scryfall_id+category]` for category queries.

    3. Create `src/stores/collection.js` exporting `initCollectionStore()`:
    - Alpine.store('collection') with state: entries (array), viewMode ('gallery'|'table'|'sets'), sortBy ('name-asc' default), filters ({ colours: [], category: 'all', search: '' }), analyticsOpen (false), loading (false), massEntryOpen (false), importOpen (false), addCardOpen (false)
    - `get filtered()` getter: filters this.entries by category (if not 'all'), then by colours (if any selected -- card must contain ALL selected colours in color_identity), then by search term (if not empty, match against card name)
    - `get sorted()` getter: sorts this.filtered by sortBy value. Options: 'name-asc', 'name-desc', 'price-desc', 'price-asc', 'set-asc', 'date-desc', 'date-asc'. Price uses card.prices.eur (or card.prices.eur_foil if entry is foil). Set sorts by card.released_at or card.set.
    - `get stats()` getter: returns { totalCards (sum of quantities), uniqueCards (entries.length), estimatedValue (sum of qty * price for each entry), wishlistCount (entries where category==='wishlist' count) }
    - `async loadEntries()`: Query db.collection.toArray(), collect unique scryfall_ids, batch-fetch from db.cards.where('id').anyOf(ids), build cardMap, join entries with card data. Use Object.freeze() on card objects to prevent Alpine deep-proxying.
    - `async addCard(scryfallId, quantity, foil, category)`: Check for duplicate via db.collection.where('[scryfall_id+foil]').equals([scryfallId, foil ? 1 : 0]).and(e => e.category === category).first(). If exists, update quantity (existing.quantity + quantity). Otherwise add new entry with foil stored as 0/1 number. Call loadEntries() after.
    - `async editEntry(entryId, updates)`: db.collection.update(entryId, updates). Call loadEntries() after.
    - `async deleteEntry(entryId)`: db.collection.delete(entryId). Call loadEntries() after.
    - `async addBatch(entries)`: Takes array of {scryfallId, quantity, foil, category}, calls addCard for each in sequence. Returns {added: count}.
    - `setViewMode(mode)`, `setSortBy(sort)`, `toggleColour(colour)`, `setCategory(category)` -- simple state setters.

    4. Modify `src/stores/app.js`: Set `locked: false` for the treasure-cruise screen entry in the screens array.
