    1. Create src/services/deck-import.js:

       Export `detectFormat(text)` (per RESEARCH.md Pattern):
       ```javascript
       export function detectFormat(text) {
         const lines = text.trim().split('\n');
         if (lines.some(l => /^\/\/\s*(Commander|Companion|The 99|Sideboard)/i.test(l))) return 'moxfield';
         if (lines.some(l => /^[A-Z][a-z]+\s*\(\d+\)$/i.test(l))) return 'archidekt';
         if (lines.some(l => /^\d+x?\s+.+\s+\([A-Z0-9]+\)\s+\d+/.test(l))) return 'arena';
         return 'plaintext';
       }
       ```

       Export `parseMoxfield(text)`: Split by section headers (`// Commander`, `// Companion`, `// The 99`). Parse each line as `{qty} {name}` pattern. Return `{ commander: [], companion: [], main: [] }`.

       Export `parseArchidekt(text)`: Split by category headers (e.g., `Creatures (23)`). Ignore category names, parse all `{qty} {name}` lines into main array. Return `{ main: [] }`.

       Export `parseArena(text)`: Parse lines as `{qty} {name} ({set}) {num}`. Return `{ main: [{ qty, name, set, num }] }`.

       Export `parsePlaintext(text)`: Parse lines as `{qty}x? {name}`. Skip blank lines and lines starting with `//`. Return `{ main: [] }`.

       Export `parseDecklist(text)`: Call detectFormat, then call the appropriate parser.

       Export `async resolveDecklist(parsed, searchFn)`: For each card entry, call searchFn(name) to find matching Scryfall card. Return `{ resolved: [{ qty, name, scryfallId, card }], unresolved: [{ qty, name, reason }] }`. For Arena format, try exact match with set+num first, fall back to name search.
