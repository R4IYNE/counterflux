       - `export const DEFAULT_THRESHOLDS = { 'Ramp': 10, 'Draw': 10, 'Removal': 8, 'Board Wipe': 3, 'Lands': 36 }`
       - `export function detectGaps(analytics, thresholds = DEFAULT_THRESHOLDS, deckSize = 100)`:
         - `const scale = deckSize / 100`
         - For each `[category, threshold]` in thresholds:
           - `const scaledThreshold = Math.round(threshold * scale)`
           - If category is `'Lands'`: `count = analytics.typeBreakdown?.Land || 0`
           - Else: `count = analytics.tagBreakdown?.[category] || 0`
           - If `count < scaledThreshold`: push `{ category, count, threshold: scaledThreshold, severity: count < scaledThreshold * 0.5 ? 'critical' : 'warning' }`
         - Return gaps array sorted by severity (critical first)
