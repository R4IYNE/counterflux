       **Methods:**
       - `async fetchForCommander(commanderName)`:
         - Set `loading.edhrec = true`, `error.edhrec = false`
         - Call `getCommanderSynergies(commanderName)`
         - If result.error: set `error.edhrec = true`, `synergies = []`, `saltScore = null`, `saltLabel = ''`, show warning toast "Intelligence unavailable -- using local heuristics." (per D-03)
         - Else: set `synergies = result.synergies`, compute salt from `result.commanderSalt` using `normalizeSalt()`, set `saltLabel` based on value (0-3: 'MILD', 4-6: 'SPICY', 7-10: 'CRITICAL')

       - `async fetchCombos(deck, cards)`:
         - Set `loading.spellbook = true`, `error.spellbook = false`
         - Extract commander names: `[deck.commander_name, deck.partner_name].filter(Boolean)`
         - Extract card names: `cards.map(c => c.card?.name).filter(Boolean)`
         - Check `db.combo_cache.get(deck.id)` for cached result
         - If cache miss or stale: call `findDeckCombos(commanderNames, cardNames)`
         - If result.error: set `error.spellbook = true`, `combos = { included: [], almostIncluded: [] }`, show warning toast "Combo detection unavailable."
         - Else: set `combos = result`, cache in `db.combo_cache.put({ deck_id: deck.id, data: result, fetched_at: Date.now() })`
         - Build `comboMap`: iterate over `combos.included`, for each combo, for each piece, add combo to `comboMap[piece.name]` (array)
         - Set `loading.spellbook = false`

         - Synchronous (no API call). Call `detectGaps(analytics, this.thresholds || DEFAULT_THRESHOLDS, deckSize)`
         - Set `this.gaps = result`

         - Else: set `this.thresholds = null` (use defaults)

    - Use day-of-year as seed to pick from top candidates: `index = dayOfYear % Math.min(candidates.length, 10)` (rotates through top 10 suggestions across days, per D-16)
