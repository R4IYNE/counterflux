Export `renderTurnTimer()` returning HTML with Alpine bindings. This is a sub-component embedded in the floating toolbar.

- Stopwatch mode (per discretion): displays `MM:SS` format computed from `$store.game.timerSeconds`
  - MM = `Math.floor(seconds / 60).toString().padStart(2, '0')`
  - SS = `(seconds % 60).toString().padStart(2, '0')`
- Three controls (icon buttons, 32x32px):
  - Play/Pause: `play_arrow` when paused, `pause` when running. Calls startTimer/pauseTimer.
  - Reset: `replay` icon. Calls resetTimer.
- Label: `TIME` in label 400 text-muted above the MM:SS display
- Timer auto-pauses when `$store.game.nextTurn()` is called (handled in store)

**2. Counter panel** (`src/components/counter-panel.js`):
Export `renderCounterPanel()` returning HTML with Alpine bindings.

- Popover above toolbar "More" button: surface background, ghost-border, padding md, z-40, max-height 400px, overflow-y auto
- Header: `COUNTERS` (label 700 uppercase)
- Available counters list: `ENERGY`, `EXPERIENCE`, `TREASURE`, `MONARCH`, `INITIATIVE`, `DAY/NIGHT`, `CITY'S BLESSING`, `STORM` (per GAME-06 / UI-SPEC)
- For each player in `$store.game.players`:
  - Player name header (label 700, player colour accent)
  - Grid of counter toggle buttons (label 400). Click to toggle counter on/off for that player: `$store.game.toggleCounter(playerIndex, counterName)`
  - For active counters, show value with +/- buttons: `$store.game.adjustCounter(playerIndex, counterName, +1/-1)`

  - `MONARCH` and `INITIATIVE`: toggle-only (no numeric value needed; only one player can have it at a time). When toggling on for one player, toggle off for all others. Implement in the counter panel click handler, not in the store.
  - `DAY/NIGHT`: global toggle, not per-player. Show as a single toggle button at the top of the panel. Display current state: `DAY` or `NIGHT`.
- Counter buttons: label 400, surface-hover bg when inactive, primary bg when active
