**1. Game stats utility** (`src/utils/game-stats.js`):
Pure functions, no Alpine dependency. Export `computeGameStats(games)` that takes an array of game records and returns:

```javascript
{
  totalGames: number,
  wins: number,
  winRate: number,  // 0-100 percentage
  avgGameLength: number,  // minutes
  avgTurns: number,
  bestDeck: { name: string, winRate: number, gamesPlayed: number } | null,
  mostPlayed: { commanderName: string, gamesPlayed: number } | null,
  currentWinStreak: number,
  longestWinStreak: number,
  winRateByDeck: [{ deckId, commanderName, wins, games, winRate }],
  winRateByPlayer: [{ playerName, wins, losses }],
}
```

Logic:
- `wins` = count where `game.winner_index === 0` (player 0 is always "you")
- `winRate` = (wins / totalGames * 100) or 0 if no games
- `avgGameLength` = average of `(ended_at - started_at)` in minutes
- `avgTurns` = average of `turn_count`
- `bestDeck` = deck_id with highest win rate (minimum 2 games), using `players[0].commander` for name
- `mostPlayed` = commander with most games
- `currentWinStreak` = count consecutive recent wins (from most recent game backwards)
- `longestWinStreak` = max consecutive wins in entire history
- `winRateByDeck` = group by deck_id, compute wins/games/winRate per deck
- `winRateByPlayer` = group opponent names across all games, count wins/losses against each

Also export `getIncrement(heldMs)`: returns 10 if heldMs >= 2000, 5 if >= 1000, 1 otherwise. This is the pure logic for long-press acceleration per D-02.

**2. Game store** (`src/stores/game.js`):
Export `initGameStore()`. Follow the `initCollectionStore()` pattern. The store has:

State:
- `view: 'setup'` -- 'setup' | 'active' | 'summary'
- `historyView: false` -- toggles between Active Game / History sections
- `selectedDeckId: null`
- `manualCommander: ''`
- `startingLife: 40`
- `opponents: []` -- array of `{ name: '', commander: '', partner: null }`
- `players: []` -- array of player objects during active game
- `currentTurn: 0`
- `timerSeconds: 0`
- `timerRunning: false`
- `_timerInterval: null`
- `gameStartedAt: null`
- `games: []` -- loaded from IndexedDB
- `expandedPlayer: null` -- index of expanded player card

Computed:
- `get stats()` -- returns `computeGameStats(this.games)`

Methods:
- `addOpponent()` -- push `{ name: '', commander: '', partner: null }` to opponents array. Max 5 opponents (6 total players).
- `removeOpponent(index)` -- splice from opponents array
- `startGame()` -- validate: must have deck or manual commander, must have >= 1 opponent. Build players array: index 0 = you (from selected deck or manual commander), indices 1+ = opponents. Each player: `{ name, commander, partner, color_index, life: startingLife, life_history: [startingLife], poison: 0, commander_damage: {}, tax_count: 0, counters: {}, eliminated: false, elimination_order: null }`. Commander damage object: keys are player indices, values are damage numbers (initialize to 0 for each other player). Set `view: 'active'`, `currentTurn: 1`, `gameStartedAt: new Date().toISOString()`. Auto-save to meta table key `active_game`.
- `adjustLife(playerIndex, amount)` -- add amount to player's life. Push new life total to life_history. Auto-save.
- `adjustPoison(playerIndex, amount)` -- adjust poison counter. Clamp to >= 0. If reaches 10, set eliminated, fire toast via `Alpine.store('toast').show()`.
- `adjustCommanderDamage(targetIndex, sourceIndex, amount)` -- adjust commander_damage[sourceIndex] on target player. Clamp >= 0. If reaches 21, fire toast.
- `adjustTax(playerIndex, amount)` -- adjust tax_count. Clamp >= 0. Tax cost = tax_count * 2.
- `adjustCounter(playerIndex, counterName, amount)` -- adjust named counter. Clamp >= 0.
- `toggleCounter(playerIndex, counterName)` -- add counter if not present (set to 0), or remove it
- `nextTurn()` -- increment currentTurn. Snapshot all players' current life into life_history. Pause timer. Auto-save.
- `startTimer()` / `pauseTimer()` / `resetTimer()` -- setInterval at 1000ms, increment timerSeconds. Store _timerInterval for cleanup.
- `endGame()` -- set `view: 'summary'`. Record `ended_at`. Clear timer interval.
- `async saveGame(winnerIndex, eliminationOrder)` -- build game record per RESEARCH.md game schema. Save to `db.games.add()`. Reload games. Clear active_game from meta. Set `view: 'setup'`.
- `async discardGame()` -- clear active_game from meta. Reset state. Set `view: 'setup'`.
- `async loadHistory()` -- load all games from `db.games.orderBy('started_at').reverse().toArray()`. Set games array.
- `async deleteGame(gameId)` -- delete from db.games. Reload games.
- `async init()` -- load games history. Check meta table for `active_game` key -- if exists, restore state (resume interrupted game).
- `_autoSave()` -- debounced write of current active game state to `db.meta.put({ key: 'active_game', ... })`. Called after every state change during active game. Use 2000ms debounce.
- `toggleExpanded(playerIndex)` -- set expandedPlayer to index if different, null if same (toggle)
