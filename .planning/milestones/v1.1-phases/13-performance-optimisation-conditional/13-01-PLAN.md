8. Repeat one more time (3 runs total).
9. Compute median across 3 runs for each metric (middle value).

12. Compute Branch decision per D-02:
    - Branch A: LCP < 2.5s AND CLS < 0.1 AND FCP < 1.0s AND INP < 200ms → ship signoff, close phase
    - Branch B: any target missed → Plans 2-6 execute
