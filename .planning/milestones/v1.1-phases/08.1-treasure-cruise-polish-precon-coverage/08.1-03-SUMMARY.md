## Decisions Made

See frontmatter `key-decisions`. Most consequential:
- **Reuse the existing card-context-menu CustomEvent** rather than introducing a parallel menu opener. Verified by reading `src/components/context-menu.js` lines 178-182 — the existing listener is source-agnostic and handles every dispatch identically. No changes required to context-menu.js.

The delegated `click` + `keydown` listeners use `data-entry-id` as the bridge from inert DOM back to the reactive `store.entries` array.

**2. [Rule 2 — Missing Critical] HTML attribute escaping for card names**
- **Found during:** Task 1 GREEN, while writing the safeName helper
- **Issue:** Plan action shows raw `${name}` interpolation into `aria-label="Quick actions for ${name}"`. A card with a quote character in its name (rare but real — e.g. "O" cards, single-letter MTG names with apostrophes in artist names if those leaked) would break the HTML attribute string and create an XSS surface for malicious bulk-data injection.
- **Fix:** Added `const safeName = String(name).replace(/"/g, '&quot;');` in `card-tile.js` before the template literal; mirrored the same escape in the virtual-scroller renderItem string in `gallery-view.js`. Used `safeName` in `aria-label`, `title`, and `alt` attributes (note: also fixes a pre-existing escape gap on `alt` in card-tile.js that was already in the original code — beneficial side-effect).
- **Files modified:** `src/components/card-tile.js`, `src/components/gallery-view.js`
- **Verification:** Test E5 still passes (asserts `aria-label="Quick actions for Lightning Bolt"` — Lightning Bolt has no quotes, so safeName is identity).
- **Committed in:** `a9f2b17` (Task 1 GREEN commit)

## Reduced-Motion @media Coordination

The plan called out a cross-plan coordination point: both Plan 01 and Plan 03 append motion-using utilities to `main.css`, and the `@media (prefers-reduced-motion: reduce)` block must not duplicate.

**Outcome:** Plan 01 ran first and extended the existing Phase 8 Plan 2 `@media` block with `.cf-panel-reopen`. Plan 03 then extended the same block to add `.card-quick-actions-checkbox` (line 158 of main.css). Final selector list:

```css
@media (prefers-reduced-motion: reduce) {
  .tc-panel-column,
  .tc-grid-column,
  .cf-panel-reopen,
  .card-quick-actions-checkbox {
    transition: none !important;
    transition-duration: 0.01ms !important;
  }
}
```

Single block, four selectors, no duplicates. ✔ Per success criteria.
